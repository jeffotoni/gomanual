Métodos - Ponteiro

Quando queremos que uma função ou método consiga modificar os dados da nossa struct, precisamos passar para ela um ponteiro, pois se passarmos somente a instância, a função irá receber uma cópia da nossa instância, portanto não conseguindo modificá-la.

A mesma regra se aplica para os receptores de métodos. 


// Essa função só fará a modificação dentro
// do próprio escopo.
func MudarNome(p Pessoa, nome string) {
    p.Nome = nome
}

// Já essa, modificará a instância em si,
// pois tem acesso ao seu endereço na memória.
func MudarNome(p *Pessoa, nome string) {
    p.Nome = nome
}

// Este também, pois o receptor é um ponteiro.
func (p *Pessoa) MudarNome(nome string) {
    p.Nome = nome
}
