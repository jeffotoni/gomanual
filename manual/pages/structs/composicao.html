Structs - Composição

Composição é um método empregado para escrever segmentos de código reutilizáveis.

Isso é obtido quando os objetos são compostos de outros objetos menores com comportamentos específicos, ou seja, objetos maiores com uma funcionalidade mais ampla são incorporados com objetos menores com comportamentos específicos. 


type Pessoa struct {
    Nome string
    idade int
}

type Programador struct {
    Linguagem string
    Pessoa
}

Quando formos instanciar uma struct que possui uma composição, precisamos também instanciar o tipo que faz a composição. Na hora da instância, simplesmente o chamamos pelo seu próprio nome de tipo.
