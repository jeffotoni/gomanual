Tipos - Error

error é um tipo de interface integrado no Go. Uma variável de erro representa qualquer valor que possa se descrever como uma string.

A interface consiste em uma única função, Error(), que retorna uma mensagem de erro de string. Go não oferece suporte para o bloco try-and-catch convencional para tratamento de erros. Em vez disso, quando algo inesperado acontece, ele retorna um erro.

Usamos a interface de erro para representar a condição de erro. Um valor nulo significa que não ocorreu nenhum erro. Para gerar erros personalizados, podemos implementar a interface de erro em uma estrutura, conforme nossos requisitos.
