Tipos - Numéricos

Um tipo numérico representa um conjuntos de valores inteiros ou de ponto flutuante.

Ponto flutuante é o nome dado às frações numéricas. 


var a int = 1 // Inteiro
var b float32 = 3.1415 // Ponto flutuante
c := 3.0
