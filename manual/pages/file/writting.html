File - Escrevendo

O ioutil.WriteFile() é usado para gravar dados em um arquivo. O método WriteFile() aceita 3 parâmetros diferentes, o primeiro é a localização do arquivo que desejamos gravar, o segundo é o objeto de dados e o terceiro é o FileMode, que representa o modo do arquivo e os bits de permissão. 


ioutil.WriteFile("path/to/file.txt", []byte("conteudo"), 0644)
