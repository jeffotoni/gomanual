Erros - Expected

Outra técnica importante em Go é definir erros esperados para que possam ser verificados explicitamente em outras partes do código. 


switch {
case errors.Is(err, ErrDivideByZero):
    fmt.Println("divisão por zero")
default:
    fmt.Printf("erro inesperado: %s\n", err)
}

Isso se torna útil quando você precisa executar uma ramificação de código diferente se um determinado tipo de erro for encontrado.
