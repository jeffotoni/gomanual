Erros - Introdução

O tratamento de erros em Go é um pouco diferente de outras linguagens de programação convencionais.

Os erros internos do Go não contêm rastreamentos de pilha, nem suportam try/catch convencional.

Em vez disso, erros em Go são apenas valores retornados por funções e podem ser tratados da mesma maneira que qualquer outro tipo de dados. 


type error interface {
    Error() string
}

O tipo de erro em Go é implementado pela interface acima.
