Erros - Criando

Erros podem ser construídos em tempo real usando o pacotes errors ou fmt. Por exemplo, a função a seguir usa o pacote errors para retornar um novo erro com uma mensagem de erro estática: 


func FazerAlgo() error {
    return errors.New("algo deu errado")
}

Da mesma forma, o pacote pode ser usado para adicionar dados dinâmicos ao erro, como um int, string ou outro erro. Por exemplo: 


func Dividir(a, b int) (int, error) {
    if b == 0 {
        return 0, fmt.Errorf("não pode dividir '%d' por zero", a)
    }
    return a / b, nil
}
